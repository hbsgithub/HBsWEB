# Openwrt 作为旁路由使用

## 客户端连接

1. 在所需要使用旁路由的设备上更改网关为 op 的 ip 地址即可
2. 如果无法访问 Google 可以更改设备 DNS 为 op 的 ip 地址或者 8.8.4.4

## 主路由连接

更改主路由 DHCP Server 的网关地址为 op 的 ip 地址即可
