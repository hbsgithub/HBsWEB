# Vim美化

:::info
首先进入vim配置文件

```bash
vim ~/.vimrc
```
在文件中添加以下内容
:::

## 1. 设置行号

```bash
set number
```

## 2. 设置语法高亮

```bash
syntax on
```

## 3. 设置配色方案

```bash
colorscheme desert
```

## 4. 设置tab键为4个空格

```bash
set tabstop=4
```

## 5. 设置自动缩进

```bash
set autoindent
```

## 6. 设置搜索高亮

```bash
set hlsearch
```

## 7. 设置搜索时忽略大小写

```bash
set ignorecase
```

## 8. 启用鼠标

```bash
set mouse=a
```
