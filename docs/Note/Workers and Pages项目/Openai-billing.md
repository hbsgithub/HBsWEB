# Openai-billing搭建

:::info

Openai-billing是一个查询Openai apikey状态的服务

:::
1. 创建Pages项目，并连接github
2. github项目地址：https://github.com/ClarenceDan/openai-billing，fork之后，将checkbilling.html重命名为index.html
3. 直接部署Pages项目即可

:::tip

任何html文件均可如此部署

:::
